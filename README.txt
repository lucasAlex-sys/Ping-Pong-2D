Comandos necessários a ser utlizado no terminal do linux(distribuição: ubunt) para compilar o jogo

sudo apt-get install make
sudo apt-get install libgl1-mesa-dev
sudo apt-get install libglew1.5-dev freeglut3-dev libglm-dev
sudo apt-get install libsoil-dev

Após as instalações, abra o terminal dentro da pasta do jogo e digite 'make' sem as aspas simples.
